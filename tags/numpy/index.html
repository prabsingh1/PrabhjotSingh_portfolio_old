Assessing Tornado Uncertainty Using Bayesian Methods

Weather Projects

This project was made for my Baysian Machine Learning final project with a few other students in my cohort. The main goal was to assess which variables are the most important in the categorization of tornado magnitude on the Enhance Fujita (EF) scale using bayesian machine learning. We used a dataset from the National Oceanic and Atmospheric Association (NOAA) to create three different models: Naive Bayes, Mulitnomial Logistic Regression and Hierarchical Logistic Regression. We found the most important variables to be the path length of the torndao, more specifically length and width of the tornado, to be the most important parameters in the classifcation of tornado magnitude. The multinomial logistic regression model performed the best with an accuracy of ~60%.
